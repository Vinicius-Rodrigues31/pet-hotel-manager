# Pet Hotel Manager - Ideia e Motivação

## Nome do Projeto

Pet Hotel Manager

## Contexto

A ideia do projeto surgiu a partir da observação do funcionamento de um hotel de hospedagem para cães onde meu próprio cachorro costuma ficar hospedado quando minha família viaja.

Ao longo do tempo, percebi que grande parte da gestão do hotel é realizada manualmente, utilizando agendas físicas e mensagens de WhatsApp para controle de reservas, consulta de disponibilidade e registro de informações dos pets.

Quando um tutor deseja hospedar seu pet, normalmente é necessário entrar em contato pelo WhatsApp para verificar se existem vagas disponíveis para o período desejado. A confirmação depende da consulta manual da agenda e do controle realizado pelos responsáveis do estabelecimento.

Essa observação despertou o interesse em desenvolver uma solução que pudesse digitalizar e automatizar parte desse processo, oferecendo mais organização para o hotel e mais praticidade para os tutores.

## Problema

Atualmente, o processo de reserva e gerenciamento de hospedagem é realizado manualmente.

Isso pode gerar dificuldades como:

- Controle manual das vagas disponíveis
- Dependência de consultas por WhatsApp
- Dificuldade para acompanhar histórico dos pets
- Organização descentralizada das informações
- Maior risco de erros operacionais

## Solução Proposta

Desenvolver uma plataforma web para gerenciamento de hospedagem de pets.

O sistema permitirá que os tutores:

- Criem uma conta
- Cadastrem seus pets
- Consultem disponibilidade de vagas
- Solicitem reservas
- Acompanhem reservas realizadas

O sistema também permitirá que os administradores:

- Gerenciem reservas
- Controlarem a ocupação do hotel
- Consultarem histórico de hospedagens
- Visualizarem informações dos pets cadastrados

## Objetivo de Aprendizado

Este projeto será utilizado para praticar e demonstrar conhecimentos em:

- Modelagem de Banco de Dados
- Oracle Database
- SQL
- PL/SQL
- Oracle APEX
- Git e GitHub
