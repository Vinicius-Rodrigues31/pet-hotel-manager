# Requisitos do Sistema

## Requisitos Funcionais

### RF01 - Cadastro de Tutor

O sistema deve permitir que um tutor realize seu cadastro.

### RF02 - Login

O sistema deve permitir autenticação através de e-mail e senha.

### RF03 - Cadastro de Pets

O tutor poderá cadastrar um ou mais pets.

### RF04 - Informações de Vacinação

O sistema deverá armazenar informações sobre a vacinação do pet.

### RF05 - Observações do Pet

O sistema deverá permitir registrar observações e cuidados especiais do pet.

### RF06 - Solicitação de Reserva

O tutor poderá solicitar uma reserva para um período desejado.

### RF07 - Aprovação de Reserva

O administrador poderá aprovar ou recusar uma solicitação de reserva.

### RF08 - Controle de Ocupação

O sistema deverá controlar a quantidade máxima de pets hospedados.

### RF09 - Histórico de Hospedagem

O sistema deverá permitir consultar o histórico de hospedagens de cada pet.
